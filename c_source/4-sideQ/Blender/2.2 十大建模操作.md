-四大操作: 挤出/内插/循环切割/倒角
E = 挤出并移动(挤出点= 线, 挤出线 = 面), 撤销只取消移动, 挤出仍然存在
CTRL + 右键 = 连续挤出

I(i) = 内插/向内挤出

CTRL + B = 倒角, 鼠标滚轮 = 增加/减少细分

ctrl + R = 循环切割, 滑动滚轮 = 增加/减少切割数量  

shift + D = 复制并移动

挤出流形 = 向里压, 没有多余的面

-六小操作: 合并/断开/填充/切刀/桥接/分离
M = 合并
V = 断开
F = 填充
K = 切刀，鼠标右键退出
CTRL + E = 桥接
P = 分离
