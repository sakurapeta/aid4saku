1. 嫉妒产生与比较，执着于他人，被他人左右，评判标准在别人身上。
2. 坚强一点，“不比较”，“不期待”，“不批评”。*把精力集中在自己身上。* 清楚自己想要什么，明白自己拥有什么。清晰认识自己，不能不劳而获。
3. 意识到生命的短暂
	1. 停止无穷尽的比较，
	2. 规避诱使的时间浪费，
	3. 时常叩问自己想要什么，觉察当下
	4. 想到就去做，有些事情不能等
4. 做自己喜欢又有时代价值的事情
	1. 自己真正喜欢的，而非建立在别人头上
	2. 具有时代价值的，而非社会强迫的
	3. 对于这样的事情要主动分配时间，分配资源
5. 头脑清晰，理性分析自己的负面想法，与自己对话，放下嫉妒的执念
